# authsys
A brief full-stack authentication project using Angular as frontend and Django Rest Framework as backend, with JWT authentication stored in HTTP-only cookies.

# Features :

- User registration and login.
- JWT authentication with cookies.
- Authenticated user endpoint.
- User logout.
- Angular Reactive Forms validation.
- Django REST API with custom User model.
- CORS enabled for frontend-backend communication.

# Requirements :

- Python
- Django
- Django Rest Framework
- PostgreSQL
- Node.js
- Angular CLI

# Run the Project :
- Backend: run `python manage.py runserver` at [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
- Frontend: run `ng serve` at [http://localhost:4200/](http://localhost:4200/)
